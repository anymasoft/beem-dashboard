# Beem Dashboard

**Веб-панель управления AI-видео-студией Beem: заказы, генерация, батч-обработка.**

Панель на Next.js для продукта Beem (AI Video Studio): создание и отслеживание заданий генерации, пакетная обработка через воркер, управление контентом. Часть экосистемы Beem (см. также `ai-video-studio`).

---

## Возможности

- **Управление заданиями** генерации видео.
- **Батч-обработка** через фоновый воркер (`worker:batch`).
- **Современный UI** на shadcn/ui + Tailwind.

---

## Стек

Next.js (App Router) · React · shadcn/ui · Tailwind CSS

---

## Быстрый старт

```bash
npm install
cp .env.example .env      # ключи провайдеров и параметры бэкенда
npm run dev
```
